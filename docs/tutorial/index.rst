Welcome to the ZyncIO tutorial!


Getting Started
===============

Install `zyncio` using your favorite Python package manager.

ZyncIO is built with type checking and developer experience as a priority. We recommend using an editor with a
good Python language server, such as Visual Studio Code with Pylance.

.. attention::
    As of the time this documentation was written, PyCharm's built-in type analyzer does not fully support all
    of the typing features used by ZyncIO. If you're using PyCharm, we recommend using the Pyright plugin.
